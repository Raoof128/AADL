# Security Policy

## Reporting

If you discover a security issue, please email `security@example.local` with details. We aim to respond within 72 hours.

## Scope

- This project is a **simulated lab**. There is no connection to production AD or PKI systems.
- All reported issues should relate to the lab code, documentation, or automation.

## Security Posture

- No default credentials are included; sample accounts exist only in configuration files.
- External network access is not required to use the lab.
- Sensitive data (certificates, keys) are placeholders and must never be used in production.
